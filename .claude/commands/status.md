# Status Command

Show the current status of the Carlsbad Chamber project.

## Check:

1. **Git Status** - Show uncommitted changes and current branch
2. **Build Status** - Check if frontend builds successfully
3. **Server Status** - Check if MCP server is running
4. **API Health** - Test Atlas API connection
5. **Pending Approvals** - Show count of pending approvals and questions
6. **Recent Activity** - Show recent git commits
7. **Environment** - Verify all required environment variables are set

Provide a comprehensive dashboard-style summary of the project health.
